Windows系统安装及BIOS设置

一、安装操作系统 #

新机组装完成，确保硬件正常启动后，就可以安装操作系统了。

可以直接使用U盘创建安装介质直接安装，最简单直接。也可以用PE系统辅助安装，但觉得没有必要。

  1. 准备一个8GB容量以上的U盘。

  2. 在另一台Windows电脑上，创建Windows11安装媒介，选择U盘为介质，等待程序制作完成。（此过程会格式化U盘）

  3. 将制作好的U盘插入新机主板上的USB接口，开机，进入BIOS，调整开机启动项为UEFI的U盘，保存重启电脑。

  4. 开机后自动进入Windows系统安装页面。进行分区操作，选择系统盘安装位置，等待安装完成。完成后自动重启，此时拔掉U盘，让系统从硬盘启动。

  5. 开机后自动进入Windows系统初始化页面。系统安装完成。

二、安装驱动软件 #

系统安装完成后，接着进行驱动软件的安装。Windows系统会自己更新基本的驱动，但一些主要的驱动，如显卡驱动，网卡驱动等，最好手动安装一下。

安装驱动软件先去主板厂商提供的技术支持官网去下。

以华硕为例，从华硕下载中心下载主板对应的驱动程序，主要包括以下几类：

  • 芯片组驱动
  • 核显驱动
  • 有线网卡驱动
  • 无线网卡驱动
  • 蓝牙驱动
  • 声卡驱动
  • USB驱动

然后安装独立显卡驱动。

  • 英伟达NVIDIA显卡驱动下载
  • AMD显卡驱动下载

三、更新BIOS #

新机可以更新一下最新的BIOS，以便有更好的硬件兼容性，首次更新后，再设置超频等，以免BIOS更新重置默认设置。

BIOS更新有多种方式，找主板厂商的技术支持文档。以华硕主板为例，可以通过EZ Flash功能更新BIOS：

  1. 准备一个FAT32格式，单一扇区，1GB容量以上的U盘。
  2. 从华硕下载中心下载主板型号所对应的BIOS文件，解压后，将CAP文件直接放置在U盘的根目录下。
  3. 电脑开机，按Delete键进入BIOS。在BIOS主页面按F7选择进入高级模式，选择Tool页面，选择ASUS EZ Flash 3 Utility，选择U盘里的BIOS更新CAP文件，确认选择进行更新BIOS，自动重启后BIOS即更新完成。

四、BIOS设置（EXPO、PBO等超频设置） #

以AMD 9700X，华硕重炮手主板为例（只展示常规超频设置）

  1. 开启内存超频EXPO

在BIOS的AI Overclock Tuner 中，选择EXPO Tweaked。

  1. 开启PBO

BIOS高级模式-AMD Overclocking 同意-Precision Boost Overdrive 中，选择高级，PBO limits 选择Manual，

PPT Limit 142000mW 或 142W；

TDC Limit 110000mA 或 110A；

EDC Limit 170000mA 或 170A。

（也可以开启TDP to 105W选项替代上述操作，但可能不稳定，需等待BIOS更新解决问题）

在Curve Optimizer 中设置全核负压20。

  1. 其他设置

如关闭关机后灯光常亮；关闭CPU核显；关闭关机后接口供电等设置。

新机的基本设置就这些，后面就是正常使用安装基本软件了。此外需要注意，最新的Edge浏览器一些设置会显著影响性能，在浏览器设置中，性能选项，系统性能下，能关闭选项尽量关闭，且注意一下游戏助手选项，关闭掉里面打开的网页，或者删除掉Game bar组件。
